Webpack打包基本用法

先写好基本代码

1.在项目所在目录运行npm install webpack webpack-cli -D命令（安装webpack相关包）

2.在项目根目录中，创建 webpack.config.js的webpack配置文件

3.在webpack配置文件中，初始化如下基本配置

module.exports={

mode:’development’ //mode用来配置编译模式（两个值可选：development【开发模式】production【生产模式】）

}

4.在package.json配置文件中的scripts节点下，新增如下代码

“scripts”:{

​ “dev”:”webpack” //scripts节点下的脚本，可以通过 npm run … 运行

}

5.在命令行运行npm run dev 命令，启动webpack进行项目打包

之后会在项目目录中多出一个dist文件夹，里面有个main.js文件，就是打包好的兼容的js文件，在html代码中引入即可。（HTML文件中不用引入原本的js文件，只需引入打包好的main.js文件）
